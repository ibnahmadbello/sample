# sample
# A simple REST to add student score and get student score.

## To get student scores: 
localhost:8080/records

## To add student record:
localhost:8080/addRecord

The request body is of the form:
 {
    "id": 10,
    "name": "Eze",
    "subject1": 75,
    "subject2": 34,
    "subject3": 85,
    "subject4": 29,
    "subject5": 45    
 }
